## 🎮 遊戲簡介

《吉伊卡哇：像素大討伐》是一款以吉伊卡哇 IP 包裝的像素風捕魚機遊戲。
操控可愛的吉伊卡哇、小八、烏薩奇，揮舞討伐棒消滅怪物，累積勞動值觸發 Bonus，挑戰強大的 BOSS！

## 🕹️ 基本操作

### 射擊
- **點擊畫面任意位置** → 向點擊方向發射討伐棒劍氣
- 每次射擊消耗對應投注等級的金幣
- 金幣不足時無法射擊

### 鎖定目標
- **點擊目標物** → 鎖定該目標，之後的攻擊自動瞄準
- 點擊 **🔓 LOCK** 按鈕 → 解除鎖定
- 鎖定中按鈕顯示 **🔒 LOCK**（黃色）

### 自動攻擊
- 點擊 **AUTO** 按鈕 → 開啟自動攻擊（按鈕變綠）
- 自動模式會智慧選擇最佳目標（優先高倍率、低 HP、快要離開的目標）
- 再次點擊關閉自動攻擊

### 切換投注
- **BET -** / **BET +** 按鈕調整投注等級（LV1-10）
- 投注等級決定使用的角色和每次射擊消耗
- Bonus Game 進行中無法切換投注

## ⚡ 勞動值系統

畫面左上角的 **💪 勞動值** 條（0-100）：

- 每次擊破目標獲得勞動值
- **勞動值滿 100** → 觸發 **Bonus Game（瘋狂拔草）**
- 勞動值接近滿時顯示 **⚡** 並變黃色提示

## 🌿 Bonus Game：瘋狂拔草

### 觸發條件
勞動值達到 100 → 3 秒準備 → 進入草地場景

### 玩法
- **15 秒內** 瘋狂點擊拔草！
- 拔越多分數越高，最終換算成金幣獎勵

### 雜草種類
| 雜草 | 分數 | 特殊效果 |
|------|------|---------|
| 🌱 普通雜草 | +1 | 無 |
| 🌿 硬雜草 | +3 | 需點擊 **2 次** |
| ✨ 發光雜草 | +8 | 增加倍率加成 |
| 🌟 金色雜草 | +20 | 觸發 **金幣雨**！ |
| 😈 搗亂怪草 | -5 | 扣分，暫停 0.3 秒 |

### 獎勵計算
```
倍率 = 20x + 分數 × 0.375（最高 50x）
獎勵 = 進場投注 × 倍率
```

> 💡 **策略**：優先拔金色雜草（+20分），避開搗亂怪草（-5分）！

## 💡 進階技巧

### 最大化勞動值效率
1. 優先鎖定**金色雜草（T104）**，一次 +15 勞動值
2. 高投注等級（LV8-10）攻速快，更容易在目標消失前擊破
3. 開啟 AUTO 模式讓系統自動選擇最佳目標

### BOSS 戰最大化獎勵
1. BOSS 出現前確保金幣充足
2. 切換到最高可負擔的投注等級
3. 開啟 AUTO 模式集中火力
4. 目標是在 **50 秒內**擊殺（400x 以上）

### Bonus Game 高分策略
1. 進場前切換到高投注等級（獎勵 = 進場投注 × 倍率）
2. 優先點擊金色雜草（+20 分）
3. 避開搗亂怪草（-5 分）
4. 發光雜草（+8 分）也值得優先拔

### 補償機制
- 30 秒內沒有高倍率獎勵，系統會提高特殊目標出現率
- 如果長時間沒有大獎，耐心等待特殊目標出現！

## ❓ 常見問題

**Q：金幣不夠了怎麼辦？**
A：降低投注等級（BET -），用較少金幣繼續遊戲。

**Q：BOSS 沒打完就消失了？**
A：BOSS 有 60 秒限時，超時會自動離開，沒有獎勵。下次更快！

**Q：Bonus Game 怎麼觸發？**
A：擊破目標累積勞動值到 100，自動觸發。金色雜草（T104）最高效。

**Q：排行榜是即時的嗎？**
A：每 10 秒更新一次。

**Q：斷線了怎麼辦？**
A：遊戲會自動重連，畫面顯示「📡 連線中斷」提示，重連成功後自動恢復。

**Q：遊戲很卡怎麼辦？**
A：系統會自動降低效能等級（LOW 模式），關閉部分特效並鎖定 30 FPS，通常能改善卡頓。

**Q：BOSS 進場時畫面閃紅是正常的嗎？**
A：是的，這是 BOSS 進場警告特效，表示強大的「那個孩子」即將出現！
