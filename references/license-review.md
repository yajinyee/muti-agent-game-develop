## IP 風險評估

### 吉伊卡哇（ちいかわ）IP 問題

**現狀**：本專案使用吉伊卡哇的角色設計和名稱。

**風險**：
- 吉伊卡哇是 nagano 的原創作品，版權明確
- 未經授權的商業遊戲可能面臨版權侵權訴訟
- 日本版權法對 IP 保護嚴格

**建議行動**：
1. **內部開發/展示用途**：目前風險可接受（非商業）
2. **商業發行前**：必須取得正式授權或改用原創角色
3. **替代方案**：設計原創角色，保留遊戲機制

**當前決策**：
- 本專案目前為技術展示/內部開發用途
- 商業化前必須解決 IP 問題
- 記錄於 `memory/project-memory.md` 的風險項目

## Stable Diffusion RAIL-M 授權重點

CreativeML Open RAIL-M 授權的主要限制：

1. **禁止**：生成有害、違法、歧視性內容
2. **禁止**：用於欺騙、詐騙目的
3. **允許**：商業使用（需遵守上述限制）
4. **允許**：修改和分發模型
5. **要求**：在分發時附上相同授權

**本專案合規性**：✅ 符合（生成的是遊戲角色，無有害內容）
