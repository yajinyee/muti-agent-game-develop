## 範例：ComfyUI 連線失敗

> 以下是一個填寫範例，供參考

**日期**：2025-01-01  
**記錄人**：Sprite Generation Agent  
**嚴重程度**：🟠 重要  
**狀態**：✅ 已解決

### 問題描述
目標：透過 API 呼叫 ComfyUI 生成角色圖像  
實際發生：HTTP 請求返回 Connection Refused

### 嘗試過的方法
嘗試 1：直接呼叫 http://127.0.0.1:8188/prompt  
結果：❌ Connection Refused  
原因：ComfyUI 服務未啟動

嘗試 2：啟動 ComfyUI 後再呼叫  
結果：✅ 成功

### 教訓
- 每次使用 ComfyUI 前必須先確認服務狀態
- 在 Skill 中加入「前置條件：確認 ComfyUI 服務運行中」
